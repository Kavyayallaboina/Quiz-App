## Online Quiz App

This is a simple console-based Online Quiz Application built using Java.
The program displays multiple-choice questions, accepts user answers, and shows the final score.

## Features

- Console-based quiz

- Multiple-choice questions

- Automatic scoring

- Final result display

## Project Structure

Question class – Stores question, options, and correct answer

Main class – Runs the quiz, collects answers, and displays score

## Output

The program displays questions with options, accepts your answers, and shows the final score at the end
